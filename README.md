Bloom & Bliss is a modern and responsive flower shop website built using HTML5 and Tailwind CSS.

This project focuses on creating a clean, visually appealing, and mobile-friendly e-commerce style interface using utility-first styling with Tailwind.

It helped me understand how to rapidly build responsive layouts and design components efficiently using Tailwind CSS.

🌟 Features

🌼 Modern hero section with strong visual hierarchy

🛍️ Product showcase section

💬 Customer review section

📦 Clean and structured footer

📱 Fully responsive design

✨ Smooth hover effects and UI interactions

🎨 Utility-first styling using Tailwind CSS

🛠️ Technologies Used

HTML5

Tailwind CSS

Responsive Design Principles

Flexbox & Grid (via Tailwind utilities)

📚 What I Learned

How to use Tailwind CSS utility classes effectively

Building responsive layouts using Tailwind breakpoints

Designing reusable UI sections

Improving spacing, alignment, and typography

Developing faster with utility-first CSS

🎯 Purpose of This Project

This project is part of my journey into front-end development.
It helped me move beyond basic CSS and explore modern styling approaches used in real-world web development.
